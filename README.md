## Features

- **Directional Launching** - Launch players in any direction with configurable boost values
- **Multiple Directions** - Support for NORTH, SOUTH, EAST, WEST, diagonal directions, UP, DOWN, PLAYER_DIRECTION, or custom angles
- **Customizable Sounds** - Configure sound type, volume, pitch, and sound source per jump pad
- **Particle Effects** - Add visual particle effects with configurable type, count, speed, and offset
- **Material Detection** - Jump pads activate when players step on specific block materials, even AIR!
- **Target Modes** - Choose between targeting blocks you're looking at or your current location
- **Full Edit Commands** - Modify any jump pad property via commands
